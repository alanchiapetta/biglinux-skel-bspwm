Atalhos do BSPWM

Terminal e Launcher

Terminal (Kitty)

  • Meta + Enter: Terminal em modo janela lado a lado
  • Meta + Shift + Enter: Terminal em modo Flutuante
  • Meta + Alt + Enter: Terminal em modo Tela Cheia

Dmenu (Launcher)

  • Meta ou Alt + F1: Menu de aplicativos
  • Meta + x: Desligar/Reiniciar
  • Meta + s: Menu de PrintScreen
  • Meta + d: Pesquisar e executar (dmenu_run)

Programas

Programas principais

  • Meta + Shift + k: Lista de Atalhos BSPWM
  • Meta + Shift + f: Gerenciador de Arquivos (PcmanFM)
  • Meta + Shift + b: Navegador (Brave)
  • Meta + Shift + e: Editor de Textos (Geany)

Programas via terminal

  • Ctrl + Alt + m: Player de Música (cmus)
  • Ctrl + Alt + e: Editor de Textos (nvim)
  • Ctrl + Alt + f: Gerenciador de Arquivos (ranger)
  • Ctrl + Alt + s: Monitor de Sistema (htop)

Sistema

Atalhos do Sistema

  • Meta + C: Fechar janela
  • Ctrl + Alt + Esc: Matar janela (xkill)
  • Ctrl + Shift + q: Logoff
  • Ctrl + Shift + r: Reiniciar BSPWM
  • Meta + Esc: Recarregar atalhos

Workspaces

  • Ctrl + Alt + Esquerda: Ir ao Workspace à esquerda
  • Ctrl + Alt + Direita: Ir ao Workspace à direita
  • Meta + 1-0: Ir ao Workspace 1-10
  • Meta + Shift + 1-0: Move janela para Workspace 1-10
  • Meta + Ctrl + Shift + Esquerda: Move janela para Workspace à esquerda
  • Meta + Ctrl + Shift + Direita: Move janela para Workspace à direita

Janelas

  • Meta + Esquerda/Direita/Baixo/Cima: Muda o foco para a janela à esquerda/direita/abaixo/acima
  • Meta + Shift + Esquerda/Direita/Baixo/Cima: Troca janela de posição para esquerda/direita/abaixo/acima
  • Alt + Shift + Esquerda/Direita/Baixo/Cima: Movimenta janelas flutuantes
  • Meta + Ctrl + Esquerda/Direita/Baixo/Cima: Expande tamanho da janela
  • Meta + Alt + Esquerda/Direita/Baixo/Cima: Encolhe tamanho da janela
  • Meta + h: Divide janela na horizontal
  • Meta + v:Divide janela na vertical
  • Meta + q: Cancela divisão da janela
  • Meta + f: Alterna janela para modo Tela Cheia
  • Meta + Espaço: Alterna janela entre os modos Flutuantes e Lado a Lado
  • Alt + Tab: Alterna o foco para a próxima janela
  • Alt + Shift + Tab: Alterna o foco para a janela anterior
